\# 🧾 ASP.NET Invoice Management System



This project is a practical test application developed using \*\*ASP.NET MVC\*\*, \*\*MSSQL\*\*, and \*\*Stored Procedures\*\*. It allows users to manage products, customers, and generate invoices.



\---



\## 🚀 Features



\### 1. Product Management

\- Create new products

\- Store product details (Name, Price, Quantity, etc.)



\### 2. Customer Management

\- Add customers with basic information

\- Form validation (Required fields, email format, etc.)



\### 3. Invoice Generation

\- Sell products to customers

\- Generate invoices using \*\*RDLC / Crystal Report\*\*

\- Automatically calculate totals



\### 4. Invoice List

\- View all invoices

\- Filter/search invoices by \*\*Customer\*\*



\---



\## 🛠️ Technologies Used



\- ASP.NET MVC

\- MSSQL Server

\- EF Core

\- RDLC / Crystal Reports

\- HTML, CSS, Bootstrap, JavaScript



\---



\## 🗄️ Database



\- SQL Server (MSSQL)

\- All operations are handled using \*\*Stored Procedures\*\*

\- Tables:

&#x20; - Products

&#x20; - Customers

&#x20; - Invoices

&#x20; - InvoiceDetails
